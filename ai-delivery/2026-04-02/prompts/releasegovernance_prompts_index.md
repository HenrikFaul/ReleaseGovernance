# ReleaseGovernance prompt index

Ez a mappa a ReleaseGovernance újratervezéséhez és Phase 1 foundation munkájához tartozó releváns promptokat és végrehajtási anyagokat tartalmazza.

## Tartalom
- `releasegovernance_redesign_package.md` – elfogadott teljes redesign baseline és validált checklist-logika
- `releasegovernance_phase1_work_package.md` – validált Phase 1 work package és regresszióvédett kiinduló állapot
- `releasegovernance_phase1_route_inventory.md` – validált route és API inventory
- `releasegovernance_prompt_pack.md` – összefoglalt átadható AI prompt csomag

## Használati szabály
- csak validált kész elem pipálható
- a promptok nem írhatják felül a governance/controller/design/lessons szabályokat
- a meglévő működő release/Jira funkciók nem sérülhetnek
- a promptok végrehajtása előtt kötelező a CHANGELOG és coding lessons beolvasása
