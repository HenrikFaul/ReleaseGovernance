This package is the first concrete code-oriented foundation package for the ReleaseGovernance redesign.

## Contains
- new domain enums and entities
- release review and approval rules
- draft vs persistent state separation basics
- repository interfaces
- Supabase foundation migration
- architecture decision notes

## Important
This is staged integration material, not proof that the runtime app has already been fully switched over.
